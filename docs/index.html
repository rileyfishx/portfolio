How each project is built

Same process both times, applied to two very different datasets.

  • Design the schema

    Start from the raw CSV, identify repeated text values, and pull them into lookup tables so the main fact table stores small integer foreign keys instead of duplicated strings.

  • Build the tables

    Numbered .sql scripts create the database, tables, constraints, and indexes in dependency order — database first, lookup tables next, fact tables and foreign keys last.

  • Load the data

    Python scripts using pandas and mysql-connector-python read the source CSV or API, resolve values against the lookup tables, and insert clean rows.

  • Write the report queries

    A library of numbered report queries answers real questions — segmentation, risk scoring, standings, home/away performance — using joins, CASE logic, window functions, and CTEs.
