NHL Analytics

NHL Standings & Roster Database

A five-table MySQL database that tracks every team, daily standings snapshot, game result, and roster. Fed by a Python ingestion script that grows the dataset every day the pipeline runs, then queried with a library of report scripts covering standings, home/away splits, roster composition, and scoring trends.

The Database

Five tables: one anchor table, three fact tables, and a join table linking players to teams by season.

TableGrowsDescription
teams Fixed One row per team — the anchor table everything else joins back to. Holds team_abbrev (e.g. CAR), team name, conference, and division.
standings Daily One row per team per date: points, wins, losses, OT losses. A new snapshot is appended every day the ingest script runs.
games Per game One row per game, with home_team_id and away_team_id both pointing back to teams, plus scores, date, and game state.
players Roster One row per player — name, position, and birth date, keyed on the NHL's own player ID.
rosters Per season Join table linking players to teams for a given season — one row per player-team-season combination.

How It Was Built

Nine numbered scripts, run in dependency order, plus a Python loader that keeps the standings table current.

  • Create the database and lookup structure

    01_create_database.sql → 02_create_teams_table.sql set up nhl_data and the anchor teams table first, since every other table references it.

  • Create the fact tables

    03_create_standings_table.sql, 04_create_games_table.sql, 05_create_players_table.sql, and 06_create_rosters_table.sql build out the tables that reference teams and, in the roster's case, players too.

  • Lock down integrity

    07_add_constraints.sql and 08_add_indexes.sql add foreign keys and indexes on the columns every report query filters or joins on — team_id, game_date, standings_date.

  • Load and refresh the data

    load_nhl.py pulls current standings, schedule, and roster data, resolves it against existing rows, and inserts new games and a fresh standings snapshot — designed to be safe to re-run daily.

Report 01

Team Standings Summary

The classic league table: every team ranked by points as of its most recent standings snapshot, with win percentage and games played derived on the fly rather than stored.

Correlated subquery RANK() OVER (PARTITION BY …) ROUND() 
USE nhl_data;

SELECT
    t.team_abbrev,
    t.team_name,
    t.conference,
    t.division,
    s.standings_date,
    s.points,
    s.wins,
    s.losses,
    s.ot_losses,
    -- games played = wins + losses + OT losses
    (s.wins + s.losses + s.ot_losses)          AS games_played,
    -- win percentage = wins / games played, rounded to 3 decimal places
    ROUND(s.wins / (s.wins + s.losses + s.ot_losses), 3) AS win_pct,
    -- rank within the full league by points (ties share the same rank)
    RANK() OVER (PARTITION BY t.conference ORDER BY s.points DESC) AS league_rank
FROM standings s
JOIN teams t ON t.team_id = s.team_id
-- keep only the latest date for each team
WHERE s.standings_date = (
    SELECT MAX(standings_date)
    FROM standings
    WHERE team_id = s.team_id
)
ORDER BY league_rank;

Report 04

Team Home vs Away Win Record

For every team, home and away records side by side — a classic hockey analysis question. Two CTEs count wins and losses from each perspective, then join back to teams so every team appears even with zero games played.

WITH … CTE Conditional SUM(CASE …) LEFT JOIN + COALESCE 
USE nhl_data;

WITH home_stats AS (
    SELECT
        g.home_team_id                                         AS team_id,
        COUNT(*)                                               AS home_games,
        SUM(CASE WHEN g.home_score > g.away_score THEN 1 ELSE 0 END) AS home_wins,
        SUM(CASE WHEN g.home_score < g.away_score THEN 1 ELSE 0 END) AS home_losses
    FROM games g
    WHERE g.game_state = 'OFF'
    GROUP BY g.home_team_id
),
away_stats AS (
    SELECT
        g.away_team_id                                         AS team_id,
        COUNT(*)                                               AS away_games,
        SUM(CASE WHEN g.away_score > g.home_score THEN 1 ELSE 0 END) AS away_wins,
        SUM(CASE WHEN g.away_score < g.home_score THEN 1 ELSE 0 END) AS away_losses
    FROM games g
    WHERE g.game_state = 'OFF'
    GROUP BY g.away_team_id
)
SELECT
    t.team_abbrev,
    t.team_name,
    t.division,
    COALESCE(h.home_games,  0) AS home_games,
    COALESCE(h.home_wins,   0) AS home_wins,
    COALESCE(h.home_losses, 0) AS home_losses,
    COALESCE(a.away_games,  0) AS away_games,
    COALESCE(a.away_wins,   0) AS away_wins,
    COALESCE(a.away_losses, 0) AS away_losses,
    CASE
        WHEN COALESCE(h.home_games, 0) = 0 THEN NULL
        ELSE ROUND(h.home_wins / h.home_games, 3)
    END                        AS home_win_pct,
    CASE
        WHEN COALESCE(a.away_games, 0) = 0 THEN NULL
        ELSE ROUND(a.away_wins / a.away_games, 3)
    END                        AS away_win_pct
FROM teams t
LEFT JOIN home_stats h ON h.team_id = t.team_id
LEFT JOIN away_stats a ON a.team_id = t.team_id
ORDER BY t.team_abbrev;

Report 07

Scoring Trends by Month

Average and total goals per month across the season, plus how often the home team wins — useful for spotting whether scoring cools off as teams tighten up defensively later in the year.

Date functions GROUP BY multiple columns Conditional aggregation 
USE nhl_data;

SELECT
    YEAR(g.game_date)                                           AS game_year,
    MONTH(g.game_date)                                          AS game_month,
    MONTHNAME(g.game_date)                                      AS month_name,
    COUNT(*)                                                    AS games_played,
    SUM(g.home_score + g.away_score)                            AS total_goals,
    ROUND(AVG(g.home_score + g.away_score), 2)                  AS avg_goals_per_game,
    ROUND(AVG(g.home_score), 2)                                 AS avg_home_score,
    ROUND(AVG(g.away_score), 2)                                 AS avg_away_score,
    -- how often the home team wins that month
    ROUND(
        SUM(CASE WHEN g.home_score > g.away_score THEN 1 ELSE 0 END) / COUNT(*),
        3
    )                                                           AS home_win_rate
FROM games g
WHERE g.game_state = 'OFF'
GROUP BY game_year, game_month, month_name
ORDER BY game_year, game_month;
